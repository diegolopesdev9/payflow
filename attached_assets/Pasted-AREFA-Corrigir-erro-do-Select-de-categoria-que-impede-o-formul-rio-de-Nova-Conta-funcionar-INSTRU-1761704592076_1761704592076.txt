Console mostra erro: "A <select> must have a value prop that is not an empty string"
Isso impede o botão "Nova Conta" de funcionar.

CORREÇÃO:

LOCALIZAR o Select de categoria no formulário (aproximadamente linhas 325-342):
```typescript
<div className="space-y-2">
  <Label htmlFor="quick-category">Categoria</Label>
  <Select
    value={quickBillData.categoryId}
    onValueChange={(value) => setQuickBillData(prev => ({ ...prev, categoryId: value }))}
  >
    <SelectTrigger className="bg-white border-gray-300">
      <SelectValue placeholder="Selecione..." />
    </SelectTrigger>
    <SelectContent>
      <SelectItem value="">Sem categoria</SelectItem>
      {categories.map((cat: any) => (
        <SelectItem key={cat.id} value={cat.id}>
          {cat.name}
        </SelectItem>
      ))}
    </SelectContent>
  </Select>
</div>
```

SUBSTITUIR POR:
```typescript
<div className="space-y-2">
  <Label htmlFor="quick-category">Categoria</Label>
  <Select
    value={quickBillData.categoryId || undefined}
    onValueChange={(value) => setQuickBillData(prev => ({ ...prev, categoryId: value }))}
  >
    <SelectTrigger className="bg-white border-gray-300">
      <SelectValue placeholder="Selecione..." />

```
