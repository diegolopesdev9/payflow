PROBLEMA IDENTIFICADO:
O window.confirm mostra URL longa e código na mensagem, deixando a interface pouco profissional.

CORREÇÃO:

1. ADICIONAR imports no início do arquivo (após os imports existentes):
```typescript
import {
  AlertDialog,
  AlertDialogAction,
  AlertDialogCancel,
  AlertDialogContent,
  AlertDialogDescription,
  AlertDialogFooter,
  AlertDialogHeader,
  AlertDialogTitle,
} from "@/components/ui/alert-dialog";
```

2. ADICIONAR estado para controlar o diálogo (após os outros useState):
```typescript
const [showDeleteDialog, setShowDeleteDialog] = useState(false);
const [showMarkPaidDialog, setShowMarkPaidDialog] = useState(false);
```

3. SUBSTITUIR a função handleMarkAsPaid:
LOCALIZAR:
```typescript
const handleMarkAsPaid = () => {
  if (window.confirm('Tem certeza que deseja marcar esta conta como paga?')) {
    markAsPaidMutation.mutate();
  }
};
```

SUBSTITUIR POR:
```typescript
const handleMarkAsPaid = () => {
  setShowMarkPaidDialog(true);
};

const confirmMarkAsPaid = () => {
  markAsPaidMutation.mutate();
  setShowMarkPaidDialog(false);
};
```

4. SUBSTITUIR a função handleDelete:
LOCALIZAR:
```typescript
const handleDelete = () => {
  if (window.confirm('Tem certeza que deseja excluir esta conta?')) {
    deleteBillMutation.mutate();
  }
};
```

SUBSTITUIR POR:
```typescript
const handleDelete = () => {
  setShowDeleteDialog(true);
};

const confirmDelete = () => {
  deleteBillMutation.mutate();
  setShowDeleteDialog(false);
};
```

5. ADICIONAR os diálogos ANTES do fechamento do último </div> (antes de </div> final):
```typescript
      {/* Dialog - Marcar como Pago */}
      <AlertDialog open={showMarkPaidDialog} onOpenChange={setShowMarkPaidDialog}>
        <AlertDialogContent className="bg-white">
          <AlertDialogHeader>

            </AlertDialogDescription>
          </AlertDialogHeader>
          <AlertDialogFooter>

            </AlertDialogAction>
          </AlertDialogFooter>
```
